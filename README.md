### 2 代码上传

- 在仓库[Mikkey-f/DigitalKids](https://github.com/Mikkey-f/DigitalKids)，创建自己的开发分支，例如创建一个mikkeyf的分支，每完成一个功能就pr到main分支，如果main分支更新了pull到自己分支里，不懂的自行百度
- 每次完成一个功能的时候在飞书上面写开发文档，比如写明这段代码为什么这么写，然后把代码cv上去
